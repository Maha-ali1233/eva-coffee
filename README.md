# Eva Coffee

A portfolio website for **Eva Coffee**, an artisan café in Greenwich Village, NYC.

## Stack

- React (Vite)
- Tailwind CSS
- Three.js / React Three Fiber
- React Router
- Framer Motion

## Pages

- **Home** — Full landing page with scroll-driven 3D coffee cup
- **About** — Minimal story and team
- **Contact** — Hours, address, and contact form

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```
